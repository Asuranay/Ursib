# Ursib
В скрипте использовал pandas т.к. скорее всего данных может быть очень много.
Нет желания ждать пока python продумает все за несколько дней.

## Установка
``pip install -r requirements.txt
``
## Запуск проекта
Создать файл .env с содержимым

    DB=используемая_база_данных
## Прописать в консоле
``python3 run.py название_файла.xlsx``

В данном примере .env не добавлен в .gitignore
